stud2

如何連線stud2.cycu.org並執行動態網站?

1.首先要開啟cmd(小黑窗)中執行ssh cd(學號)@stud2.cycu.org

 *若要用可攜中start.bat中的cmd要先將start.bat用編譯器打開並將第三行蓋掉,然後重新啟動。

2.執行完會要你打密碼,密碼在學校Mail2000,3/8的一封stud2.cycu.org 帳號通知信裡。

3.連線完會進到OpenSSH SSH client的黑窗,再來是要生成ssh的key,在黑框中執行ssh-keygen。

4.接下來就是要用filezilla連線stud2.cycu.org將.ssh目錄下的id_rsa.pub拉到本地的目錄中,

再用編譯器打開,並將內容送入到Github->Settings->SSH and GPG keys。

5.然後下載config並用filezilla送進.ssh目錄下

6.然後在OpenSSH SSH client中執行以下三個指令:

    1.git config --global user.name "Github帳號名"

    2.git config --global user.email "Github信箱"

    3.git config --global http.proxy http://p42.cycu.org:3128

7.將遠端Github的倉儲克隆下來,有以下兩種方法:

    1.http協定:git clone https://github.com/mdecd2023/2a-pj1agx.git

    2.ssh協定:git clone git@github.com:mdecd2023/2a-pj1agx.git

*由於克隆的方式不一樣倉儲中.git中的config設定也不同,因為最後是要用ssh推送到Github,用https克隆的要將config設定改成ssh的樣子,也就是將https://github.com/mdecd2023/2a-pj1agx.git改成git@github.com:mdecd2023/2a-pj1agx.git

8.再來要下載server.py的檔案用編輯器將自己9開頭的埠號輸入並將檔案用FileZilla送到倉儲目錄下

9.然後在小黑窗執行cd 2a-pj1agx移動到倉儲目錄下執行python3 server.py(沒出現東西就算成功)

10.在firefox或waterfox中執行https://stud2.cycu.org:8xxx

*若有設定proxy要關掉
